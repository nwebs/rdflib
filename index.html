Release Notes

This release is a bug fix release and is the second release since the World Wide Web Consortium Issues RDF and OWL Recommendations. The release conforms to the RDF/XML Syntax Specification (Revised) and passes all of the positive and negative RDF Test Cases.

RDFLib is back to a single branch of development (as of version 2.0.0). The overhead of having two branches, a stable and a development one, is no longer needed.
