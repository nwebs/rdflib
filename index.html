Release Notes

This release updates the parser to follow the October 10, 2003 working drafts and contains a few bug fixes. The release is a development release in the development branch. See also the RDF/XML Syntax Specification (Revised). As of release 1.2.1 RDFLib now passes all of the approved positive and negative rdfcore parser test cases.

Releases in the development branch are for developing new functionality. The releases will likely be less stable and tested from release to release within this branch than in the stable branch.
