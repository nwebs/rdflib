Release Notes

This release is a candidate release... the plan is for it to become the next release on Monday (2004/03/15). Please feel free to try out this candidate release and report bugs to help make the next release bug free.

This release is the first release since the World Wide Web Consortium Issues RDF and OWL Recommendations. The release conforms to the RDF/XML Syntax Specification (Revised) and passes all of the positive and negative RDF Test Cases.

As of this release, RDFLib will go back to a single branch. The overhead of having two branches, a stable and a development one, is no longer needed.
